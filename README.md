# XML Utility
Project developed as laboratory work 2 for OOP subject.
## Use cases:
* Convert .XML to .HTML
* Find in .XML by keywords using DOM or SAX
## Table contents:
.XML file consists of the following attributes:
```
Name: Firstname, Surname, Lastname; 
Workplace: Faculty, Department; 
Education: Basic, Additional, Postgraduate, Doctoral.
```
